## 파이썬 가상 환경 구성

> VS Code 기준, `Ctrl + Shift + P` 로 명령 팔레트를 열어 'Python 환경 만들기' 메뉴를 선택하여 venv를 구성하세요.


<br/>

## 파이썬 가상 환경 설치

Python 인터프리터 버전: 3.11.2


```bash
pip install -r requirements.txt
```
